Notation

  • Classc ≈ context (a general whole).
  • Instancei ≈ content (a specific part).
  • Ic = class of the immanent
  • Ii = instance of the immanent
  • Oc = class of the omniscient
  • Oi = instance of the omniscient
  • Tc = class of the transcendent
  • Ti = instance of the transcendent

Based on the axioms of An Immanent Metaphysics.

  • The Immanent informs the Omniscient.
  • The Omniscient serves the Transcendent.
  • The Transcendent orients to the Immanent.

...an instance of the immanent (which implies a class of the immanent), precedes an instance of the omniscient (which implies a class of the omniscient), precedes an instance of the transcendent (which implies a class of the transcendent), precedes...

The Triadic Loop

  • Beauty informs Truth (Ii ⇒ Ic → Oi ⇒ Oc)
  • Truth serves Goodness (Oi ⇒ Oc → Ti ⇒ Tc)
  • Goodness orients to Beauty (Ti ⇒ Tc → Ii ⇒ Ic)

...an instance of Beauty (which implies a class of Beauty), precedes an instance of Truth (which implies a class of Truth), precedes an instance of Goodness (which implies a class of Goodness), precedes...
