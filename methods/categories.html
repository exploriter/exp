--- title: Post Categorization (How-to) tags: ["method", "info"] description: How to categorize a new post on exploriter.com. ---

1. What is the primary purpose of the post?

  • Knowledge
  • To explain what something is → Concept
  • To show how to do something → Method
  • To tell why, where, when, and/or who through narrative context → Story
  • Activity
  • To describe a new open-ended exploration → Playground
  • To describe a finite goal-directed initiative → Project
  • To describe a meaningful recurring discipline → Practice

2. If still unclear, ask:

  • Does it transform understanding? → Concept
  • Does it transform ability? → Method
  • Does it transform perspective? → Story
  • Does it transform doing? → Playground or Project
  • Does it transform being? → Playground or Practice

3. If still unclear, what makes up most of the content?

  • Definitions, models, principles → Concept
  • Steps, techniques, instructions → Method
  • Narratives, contexts, examples → Story
  • Experiments, ideas, reflections → Playground
  • Objectives, plans, artifacts → Project
  • Disciplines, habits, rituals → Practice
